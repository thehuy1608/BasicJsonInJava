# Basic JSON manipulation in Java

IDE: NetBeans 8.2

JDK version: jdk1.8.0_152

API: Gson API 2.8.2

Some simple ways to serialize and deserialize JSON File in Java using Gson API.

Add Gson using Gradle:

dependencies {
    compile 'com.google.code.gson:gson:2.8.2'
}
